== Installation ==

1. Go to the Add New plugins screen in your WordPress Dashboard.
2. Click the upload tab.
3. Browse for the plugin file (epic-calculator.zip) on your computer.
4. Click "Install Now" and then hit the activate button.
5. Paste [epic_cc_calculator] shortcode in the page where you want the Credit Card Charges Calculator.
6. Paste [epic_comp_int_calculator] shortcode in the page where you want the Compound Interest Calculator.
